DNS Security Architecture

System-level protection with browser freedom

🔒 Layer 1: System DNS (Enforced)

  • CleanBrowsing Family Filter via DNS-over-TLS
  • Blocks malware, adult content, phishing
  • Covers all OS apps, CLI tools, services
  • Fallback: Cloudflare Family (1.1.1.3)

🌐 Layer 2: Browser DoH (User's Choice)

  • No managed policies — no restrictions
  • Users can configure any DoH provider
  • Default: follows system DNS settings
  • Cloudflare, Google, Quad9, or any custom
